Yodlee account aggeration demo
This is an application demo for Yodlee account  aggregation tool which demonstrates  how an account can be aggregated and how can the  aggregated data be retrieved and used.

Click the button below to open Yodlee account aggeration tool (Yodlee account aggeration tool is a pre-packaged widget connected to the Yodlee sandbox enviroment. This widget lets consumers add accounts by searching for the website of a financial institution and entering the login credentials for the account(s) held there. Consumers can add savings, checking, investment, credit card, insurance account types and many more).
